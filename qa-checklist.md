# QA checklist

Layout:
- Container width + page gutters match
- Section spacing matches
- About section becomes single column on mobile

UI:
- Pills (hero eyebrow, buttons, view-casestudy overlay) match radius + border + blur
- Image cards match corner rounding

Typography:
- Hero H1 size/weight/line-height match
- Muted paragraph opacity matches

Behavior:
- External links open correctly (Behance, booking link)
- Focus states are visible (accessibility)
